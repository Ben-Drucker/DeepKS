def make_predictions(kinase_seqs, kin_info, site_seqs, site_info, predictions_output_format='csv', suppress_seqs_in_output=False, verbose=True, pre_trained_gc=PRE_TRAINED_GC, pre_trained_nn=PRE_TRAINED_NN, device='cpu', scores=False, normalize_scores=False, dry_run=False, cartesian_product=False, group_output=False, bypass_group_classifier=[]): ¶

Make a target/decoy prediction for a kinase-substrate pair.

Parameters
kinase_seqs:list[str]The kinase sequences. Each must be >= 1 and <= 4128 residues long.
kin_info:dictThe kinase (meta-) information. See ./kin-info_file_format.txt for the required format.
site_seqs:list[str]The site sequences. Each must be 15 residues long.
site_info:dictThe site (meta-) information. See ./site-info_file_format.txt for the required format.
predictions_output_format:typing.Literal['inorder', 'dictionary', 'inorder_json', 'dictionary_json', 'csv', 'sqlite']

The format of the output.

  • inorder returns a list of predictions in the same order as the input kinases and sites.
  • dictionary returns a dictionary of predictions, where the keys are the input kinases and sites and the values are the predictions.
  • in_order_json outputs a JSON string (filename = "DeepKS/out/current-date-and-time.json") of a list of predictions in the same order as the input kinases and sites.
  • dictionary_json outputs a JSON string (filename = "DeepKS/out/current-date-and-time.json") of a dictionary of predictions, where the keys are the input kinases and sites and the values are the predictions.
  • csv outputs a CSV table (filename = "DeepKS/out/current-date-and-time.csv"), where the columns include the input kinases, sites, sequence, metadata and predictions.
  • sqlite outputs a sqlite database (filename = "DeepKS/out/current-date-and-time.sqlite"), where the columns include the input kinases, sites, sequence, metadata and predictions.
suppress_seqs_in_output:boolWhether to include the input sequences in the output.
verbose:boolWhether to print predictions.
pre_trained_gc:strPath to previously trained group classifier model state.
pre_trained_nn:strPath to previously trained neural network model state.
device:strDevice to use for predictions.
scores:boolWhether to return scores.
normalize_scores:boolWhether to normalize scores.
dry_run:boolWhether to run a dry run (make sure input parameters work).
cartesian_product:boolWhether to make predictions for all combinations of kinases and sites.
group_output:boolWhether to return group predictions.
bypass_group_classifier:list[str]List of known kinase groups in the same order in which they appear in kinase_seqs. See ./kin-info_file_format.txt for instructions on how to specify groups.
